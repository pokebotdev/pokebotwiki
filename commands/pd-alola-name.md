# pd alola \[name\]

Users can also choose to search Alolan Pokemon by name.

{% hint style="warning" %}
Using `alola` now makes Pokebot search **ONLY** Alolan Pokemon.
{% endhint %}

Example: `pb!pd alolan Rattata`

![is what Pokebot will display](../.gitbook/assets/pd-alola4.PNG)
