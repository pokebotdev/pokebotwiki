# pd alola \[dexNum\]

Users can search Alolan Pokemon by Alolan Dex Number.

{% hint style="warning" %}
Using `alola` now makes Pokebot search **ONLY** Alolan Pokemon. 
{% endhint %}

Example: `pb!pd alolan 015`

![is what Pokebot will display](../.gitbook/assets/pd-alola4.PNG)
