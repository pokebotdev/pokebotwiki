# pd mega \[name\]

Users can also choose to search Mega Pokemon by name.

{% hint style="warning" %}
Using `mega` now makes Pokebot search **ONLY** Mega Pokemon.
{% endhint %}

Example: `pb!pd mega Venusaur`

![is what Pokebot will display](../.gitbook/assets/pd-mega2.PNG)
